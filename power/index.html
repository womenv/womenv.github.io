# 女性力量

女性不是男人的老婆，不是外貌的奴隶，更不是毫无反抗能力的被害人。

女性是生机勃勃的人。

没有所谓的“女性也能”，女性本来就能。希望榜样们的故事能给读者们带来冲破枷锁的勇气和力量。

  • 邓玉娇 (opens new window)

宾馆服务员。在工作期间受到三名男性客人的性骚扰，为求自保使用修脚刀防卫，造成骚扰者一死一伤。事发时邓玉娇22岁，体格较为瘦小，仅凭一把小刀便可以保护自己，一定程度上可以说明所谓的“男女体力差”在刀面前不值一提。更耐人寻味的是，事后调查发现邓玉娇为心境障碍病人。也就是说，邓玉娇也许并不能代表广大普通女性，但她无法代表的方面不是体力，而是反抗精神。邓玉娇身为心境障碍病人，更容易在受到威胁时反抗和释放攻击性，而这也是我们广大女性长久以来受到压抑的方面。女性总是成为性别暴力新闻中下场很惨的受害人，也许所谓的“体力差异”只是很小的一个原因。

不要强调体力差异。要积极提高自己的身体素质，更重要的是，要反抗。

  • 余秀华 (opens new window)

诗人。生于农村，19岁被嫁人，22岁写了第一首诗，33岁正式开始写诗，36岁去温州打工，38岁在《诗刊》发表作品，39岁当选湖北省钟祥市作家协会副主席，同时出版诗集《月光落在左手上》 (opens new window)和《摇摇晃晃的人间》 (opens new window)，前者成为当时20年来中国销量最大的诗集。39岁离婚。部分作品 (opens new window)

  • 芮乃伟 (opens new window)

棋手。1963年生人，24岁参加中日围棋擂台赛，期间因去男棋手房间下棋被取消比赛资格，被称为三峡事件。25岁升为九段。由于三峡事件和女性身份，她在国内参加比赛的机会远少于同水平的男棋手，便去日本寻求机会。30岁拜吴清源为师。33岁，日本棋院拒绝接纳她。37岁参加韩国围棋“国手战”，击败李昌镐和曹薰铉夺冠。48岁回到中国棋院。2021年，58岁的她在第十四届全运会的围棋混双比赛中与队友夺得金牌。

注：名字的正确写法为芮迺伟，但因为误写的版本流传更广泛且易读，所以暂时使用了“芮乃伟”。

  • 火神山项目工地女性工人 (opens new window)

占火神山项目工地工人的50%。

只要你掌握话语权，就可以想怎么说就怎么说。体力活不需要女人时，可以说”女人体力不好不适合干体力活“；技术活不需要女人时，可以说”女人笨干不好技术活，但是女人可以去做体力活“。总之，只要是相对赚钱的工作，都可以说成是不适合女人的。而你总能在相对最不赚钱的工作中看到大量的女性劳动者，这无关体力。
